    Processo Administrativo

    Como devemos criar um processo administrativo

    Primeiro passo

    Ao logar no sistema deverá ser clicado no menu lateral com o nome de "Compras" Ao clicar devemos ver uma listagem de documentos de formalização de demandas Acessando um dos dfd's* deverá encontrar um botão ao lado superior direito "gerar processo"

    For full documentation visit mkdocs.org.

    Commands

    • mkdocs new [dir-name] - Create a new project.
    • mkdocs serve - Start the live-reloading docs server.
    • mkdocs build - Build the documentation site.
    • mkdocs -h - Print help message and exit.
